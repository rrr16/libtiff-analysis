# Сработка 2

**Идентификатор правила: ** arrayIndexOutOfBounds

**Уровень критичности: ** error

**Локация в коде:**

- Файл: `libtiff/tif_fax3.c`
- Строка: 601
- Фрагмент:
```
_PutBits(tif, bits, length);
```

**Оценка валидности сработки**: Не подтверждается

**Описание участка кода: ** Данный участок кода является частью реализации кодека Fax Group 3 и отвечает за упаковку битовых кодов переменной длины в поток байтов перед записью в выходной буффер TIFF. Макрос `_PutBits` поэтапно записывает биты в текущий байт, а при его заполнении выполняет сброс через `_FlashBits`.

**Потенциальный риск: ** Анализатор сообщает о возможном выходе за границы массива `_msbmask` при обращении по индексу `length`. Однако цикл внутри макроса выполняется пока `length > bit`, в теле цикла переменная `length` уменьшается на значение `bit` каждую итерацию, поэтому после завершения цикла `length` не превысит `bit`. Переменная `bit` является частью состояния кодека и принимает значения только от 1 до 8. Таким образом обращение к массиву `_msbmask` по индексу `length` никогда не выйдет за его границы.

**Рекомендация по исправлению: ** Изменения в исходном коде не требуются. 

# Сработка 3

**Идентификатор правила: ** memleakOnRealloc

**Уровень критичности: ** error

**Локация в коде:**

- Файл: `libtiff/tools/tiffdump.c`
- Строка: 250
- Фрагмент:
```
visited_diroff = (uint64_t*) realloc(visited_diroff, alloc_size);
```

**Оценка валидности сработки**: Подтверждается

**Описание участка кода: ** В функции `dump()` массив `visited_diroff` расширяется с помощью функции `realloc()`. При неудачном вызове `realloc()` возвращается `NULL` и указатель `visited_diroff` перезаписывается, старый блок памяти теряется. Это приводит к утечке памяти при невозможности выделить новый блок.

**Потенциальный риск: ** Потеря памяти при каждом неуспешном вызове `realloc()`. В реальной работе программы может вызвать рост использования памяти или аварийное завершение.

**Рекомендация по исправлению: ** Использовать временный указатель для безопасного вызова `realloc()`:
```
uint64_t* visited_diroff_tmp = (uint64_t*) realloc(visited_diroff, alloc_size);
if (!visited_diroff_tmp){
	free(visited_diroff);
	Fatal("Out of memory!");
}
visited_diroff = visited_diroff_tmp;
```
